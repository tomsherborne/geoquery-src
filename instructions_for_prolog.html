Instructions for running Prolog queries for Geoquery Data

  1. Load up Sicstus Prolog(tm).

  2. Prove compile('geoquery').

  3. To try executing an actual query, try to prove this predicate:
execute_query(<Query>, Ans).
where <Query> is a parsed query in logical form (you can grab an example from "geoqueries880") and the set of answers is returned inAns.
